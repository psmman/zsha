# Chezmoi Plugin for Oh My Zsh

## Introduction

The `chezmoi plugin` adds shortened aliases for common chezmoi commands to Oh My Zsh.

To use it, add `chezmoi` to the plugins array of your zshrc file:


```bash
plugins=(... chezmoi)
```
